# Drop Number Game

This is a **Data Structures** course project developed in Java.

## 🎮 About the Game
Drop Number Game is a puzzle/strategy game where numbers drop from the top of the grid. The main goal is to strategically place and merge identical numbers to achieve higher scores and prevent the board from overflowing.

## 🛠️ Tech Stack & Data Structures Used
* **Language:** Java
* **Concepts:** Object-Oriented Programming (OOP)
* **Data Structures Used:** 
  * [MultiLinked List]

## 🚀 Project Structure
* `src/`: Contains all the main Java source code files.
* `build.xml`: Ant build script for compiling and running the project.
